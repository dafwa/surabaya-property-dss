# surabaya-property-dss
Sistem Pendukung Keputusan untuk pemilihan properti di wilayah Surabaya menggunakan metode hybrid Simple Additive Weighting (SAW) dan Technique for Order of Preference by Similarity to Ideal Solution (TOPSIS). Dikembangkan sebagai proyek akademik untuk mata kuliah Sistem Pendukung Keputusan.
